Hugging Face 空间部署教程

在Hugging Face平台上创建和部署您的AI应用空间

部署步骤

打开官方网站：https://huggingface.co

详细注册和完整版部署教程【点击查看】

（比本期部署方法更加稳定耐用）

登录后创建空间

创建空间时 Space SDK 选择 > Docker > JupyterLab

注意： JupyterLab 是 Jupyter 项目推出的下一代基于 Web 的交互式开发环境，旨在替代传统的 Jupyter Notebook，提供更强大的功能和更灵活的用户体验。它支持多种编程语言和文件类型，广泛应用于数据科学、机器学习、科学计算和教育等领域。

然后你就可以在这个空间运行各种代码了。

部署命令

复制并执行以下命令来部署您的应用：

bash <(curl -l -s https://raw.githubusercontent.com/zzzhhh1/free-vps-py/refs/heads/main/test.sh)

提示： 点击上方的"复制命令"按钮，然后将命令粘贴到您的终端中执行。

使用提示

  • 确保您已登录 Hugging Face 账户
  • 创建空间时选择正确的硬件配置
  • 部署过程可能需要几分钟时间
  • 部署完成后，您的应用将有一个公开的URL
