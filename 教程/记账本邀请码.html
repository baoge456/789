家庭记账本 - 用户注册邀请码

邀请码说明

什么是邀请码？

  • 邀请码是注册家庭记账本账户的必需凭证
  • 每个邀请码只能使用一次
  • 邀请码系统有效防止恶意注册和滥用
  • 确保只有授权用户才能使用系统
  • 所有用户都是普通用户，享受相同的记账功能

重要提醒

请妥善保管您的邀请码，不要分享给不信任的人。每个邀请码都是宝贵的资源，使用后无法重复使用。

普通用户邀请码

用于注册普通用户账户，可以正常使用所有记账功能

baoge666
baoge777
baoge888

邀请码位置

文件位置：

  • src/simple-index.js - 后端API文件（第54行）
  • 邀请码说明.html - 当前说明页面

具体位置：

  • 在 src/simple-index.js 第54行：const validInviteCodes = ['baoge666', 'baoge777', 'baoge888'];
  • 在 src/simple-index.js 第55行：const adminInviteCodes = ['ADMIN2024', 'SUPERADMIN2024'];
  • 邀请码验证逻辑在第56-65行

如何修改邀请码

修改步骤：

  1. 打开 src/simple-index.js 文件
  2. 找到第54行的 validInviteCodes 数组
  3. 修改数组中的邀请码值
  4. 同时更新 邀请码说明.html 中的显示
  5. 保存文件并重新部署到Cloudflare Workers

注意事项：

  • 修改后需要重新部署到服务器
  • 确保新旧邀请码格式一致
  • 建议保留一些备用邀请码
  • 修改前请备份原文件

技术说明

代码中的邀请码位置

在 src/simple-index.js 中的具体位置：

  • 普通用户邀请码： 第54行 validInviteCodes 数组
  • 管理员邀请码： 第55行 adminInviteCodes 数组
  • 验证逻辑： 第56-65行的 if 语句

修改方法：

  • 直接修改第54行的 validInviteCodes 数组
  • 确保格式保持一致（字符串数组格式）
  • 修改后需要重新部署到Cloudflare Workers
  • 测试新邀请码是否有效

重要提醒

  • 修改邀请码后必须重新部署才能生效
  • 已使用的邀请码无法重复使用
  • 建议定期更换邀请码以提高安全性
  • 修改前请确保理解代码逻辑

使用说明

注册步骤

  • 访问家庭记账本网站
  • 点击"注册新用户"
  • 填写用户名、密码和确认密码
  • 输入邀请码（从上方选择）
  • 点击注册按钮完成注册

安全建议

  • 使用强密码（至少6位字符）
  • 不要使用容易猜测的用户名
  • 定期更换密码
  • 不要在公共设备上保存登录状态
