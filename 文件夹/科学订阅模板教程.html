Cloudflare Worker 书签管理部署教程

简介

这个 Cloudflare Worker 提供了一个功能丰富的书签管理页面，支持分类、搜索、管理员登录、数据导入导出、拖拽排序等功能。

核心功能亮点

  • 响应式设计： 适配桌面和移动设备。
  • 分类管理： 方便地组织您的书签。
  • 站内搜索： 快速找到所需书签。
  • 管理员模式： 密码保护，可编辑、添加、删除、排序书签和分类。
  • 数据持久化： 使用 Cloudflare KV 存储数据。
  • 数据导入/导出： 支持 JSON 格式，方便备份和迁移。
  • 公共访问密码：： 可选设置，保护您的公共页面。

部署步骤

1. 登录 Cloudflare 并创建 Worker

  1. 登录您的 Cloudflare 账号。
  2. 在左侧导航栏中，点击 Workers & Pages。
  3. 点击 创建应用程序。
  4. 点击 创建 Worker。
  5. 为您的 Worker 命名（例如：my-bookmarks-worker），然后点击 部署。

2. 获取并粘贴 Worker 代码

  1. Worker 部署成功后，点击 编辑代码。
  2. 将默认的代码全部删除。
  3. 点击下方链接，在新页面中复制代码：
    点此获取 Worker 代码

    在新打开的页面中，复制全部代码内容，并粘贴到 Cloudflare Worker 的代码编辑器中。

3. 点击 保存并部署。

3. 配置 KV 存储

为了持久化存储书签数据，您需要绑定一个 Cloudflare KV Namespace。

  1. 在 Worker 页面中，点击 设置 选项卡。
  2. 滚动到 变量 部分，找到 KV Namespace 绑定。
  3. 点击 添加绑定。
  4. 在 变量名称 字段中，输入：CARD_ORDER
  5. 在 KV Namespace 下拉菜单中，选择一个现有的 KV Namespace，或者点击 创建新的 KV Namespace 并为其命名（例如：my-bookmarks-data）。
  6. 点击 保存。

4. 配置环境变量

为了设置管理员密码和可选的公共访问密码，您需要添加环境变量。

  1. 在 Worker 页面中，点击 设置 选项卡。
  2. 滚动到 变量 部分，找到 环境变量。
  3. 点击 添加变量。

管理员密码 (ADMIN_PASSWORD)

  • 在 变量名称 字段中，输入：ADMIN_PASSWORD
  • 在 值 字段中，输入您的管理员密码。请务必设置一个强密码！
  • 点击 保存。

公共访问密码 (PUBLIC_PASSWORD) - 可选

  • 如果您希望您的书签页面在公共访问时也需要密码，请添加此变量。
  • 在 变量名称 字段中，输入：PUBLIC_PASSWORD
  • 在 值 字段中，输入您的公共访问密码。
  • 点击 保存。
  • 注意： 如果不设置此变量，或者将其值留空，您的书签页面将对所有人公开可见，无需密码即可访问。

5. 访问您的 Worker

部署完成后，Cloudflare 会为您提供一个 Worker URL (例如：your-worker-name.your-username.workers.dev)。您可以通过这个 URL 访问您的书签页面。

可选：绑定自定义域名

如果您想使用自己的域名（例如：bookmarks.yourdomain.com），可以按照以下步骤操作：

  1. 在 Worker 页面中，点击 触发器 选项卡。
  2. 在 自定义域 部分，点击 添加自定义域。
  3. 输入您想要绑定的域名，然后按照提示完成 DNS 配置。

如何使用

1. 公共访问：

  • 直接访问 Worker URL 或自定义域名。
  • 如果设置了 PUBLIC_PASSWORD，会先显示一个登录页面。

2. 管理员模式：

  • 在页面顶部，长按标题（例如：“科学订阅”）约 0.7 秒。
  • 页面将切换到管理员登录界面。
  • 输入您设置的 ADMIN_PASSWORD 登录。
  • 登录成功后，您会看到页面右上角出现一个返回按钮，并且页面底部会显示管理操作按钮（添加分类、添加书签、保存更改等）。
  • 在管理员模式下，书签和分类卡片上会出现编辑、删除和排序按钮。
  • 点击右上角的返回按钮可以退出管理员模式，回到公共视图。

自定义名称和 Logo

您可以轻松地将“科学订阅”更改为“书签管理”或其他您喜欢的名称，并更换 Logo。

1. 回到 Cloudflare Worker 的代码编辑界面。

2. 修改 Logo 链接：

  • 在代码编辑器中，使用 CTRL + H (Windows/Linux) 或 CMD + F (macOS) 打开查找功能。
  • 在检索框中输入：科学订阅 Logo
  • 查找结果会定位到 Logo 的 alt 属性。在该位置附近，您会找到 href 属性，其值是 Logo 图片的 URL。
  • 将 href 属性的值替换为您自己的 Logo 图片 URL。通常会有 2 处需要更换（公共访问页面和主页面）。

3. 修改名称：

  • 同样使用 CTRL + H (Windows/Linux) 或 CMD + F (macOS) 打开查找功能。
  • 在检索框中输入：科学订阅
  • 在替换框中输入您想要的新名称，例如：书签管理。
  • 点击“全部替换”按钮（或类似功能）进行批量替换。

4. 完成所有修改后，点击 保存并部署 您的 Worker。
