Hexo博客文章加密码保护教程【vs code】

1. 安装 hexo-blog-encrypt 插件

npm install hexo-blog-encrypt --save

2. 配置 _config.yml

在 Hexo 站点根目录下的 _config.yml 文件中添加如下内容：

encrypt:
  abstract: 这是一篇加密文章，输入密码查看内容。
  message: 请输入密码
  tags:
    - 密码保护

3. 设置需要加密的文章

在你想加密的文章的头部（Front-matter）添加如下内容：

---
title: 你的文章标题
date: 2024-06-01
tags:
  - 密码保护
password: 你的密码
---

或者只需添加 password 字段：

password: 你的密码

4. 重新生成并部署博客

本地演示：hexo cl; hexo s
远程部署：hexo cl; hexo g; hexo d

hexo clean
hexo g
hexo d

5. 效果预览

访问加密文章时，会自动弹出密码输入框，输入正确密码后才能查看内容。

注意：如需加密多篇文章，分别在每篇文章的头部添加 password 字段即可。

如遇问题，请确认插件已正确安装，并已在 _config.yml 中完成配置。
